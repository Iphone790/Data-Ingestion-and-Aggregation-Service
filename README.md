# Data Ingestion & Aggregation Service

## Overview

This project is a backend event ingestion and analytics system built using Django REST Framework.

The system supports:
- Event ingestion
- Bulk event processing
- Idempotent event handling
- Aggregation processing
- Metrics APIs
- Concurrency-safe operations
- Automated testing

---

## Tech Stack

- Python 3.11
- Django
- Django REST Framework
- PostgreSQL
- Pytest
- Docker
